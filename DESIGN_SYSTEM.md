# Design System — Ruang Dampak

## Brand Colors
- Primary: #022f6e (navy deep)
- Accent CTA: #0ea5e9 (teal)
- Font Display: Plus Jakarta Sans
- Font Body: DM Sans

## Aturan wajib (ikuti di SEMUA prompt):
- Gunakan class dari Tailwind config extend, bukan default Tailwind
- Tombol CTA utama: bg-primary-900 text-white
- Tombol secondary: border border-primary-300 text-primary-900
- Glassmorphism hanya di atas background gradient gelap
- Border radius default: rounded-xl
- Spacing kelipatan 4px (p-4, p-6, p-8, gap-4, gap-6, dst)
- JANGAN gunakan font Inter, Roboto, Arial
- JANGAN hardcode warna di luar Tailwind config
- Semua heading: font-display font-semibold
- Semua body text: font-body
